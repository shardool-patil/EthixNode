# EthixNode 🌍🤖

**EthixNode** is an AI-powered global remittance gateway designed to bring radical transparency and predictive intelligence to international money transfers. 

Built for the **Global Fusion Hackathon 2026**, this project aims to solve the "hidden fee" and volatility challenges faced by students, expats, and families in the global economy.

---

## 🚀 The Challenge
Traditional cross-border transfers are often opaque, with banks hiding 3-7% markups in exchange rates. Additionally, users often lose money by sending funds at high-volatility moments.

## 💡 Our Solution
EthixNode provides:
* **Predictive Forecasting:** A Python-based AI service that analyzes currency trends to suggest the optimal time to send money.
* **Radical Transparency:** A Spring Boot backend that calculates "Real Market Rates" vs. "Traditional Bank Rates" to show users exactly how much they save.
* **Secure Ledger:** A robust PostgreSQL database to track global transactions and realized savings.

## 🛠️ Tech Stack
* **Frontend:** React.js, Tailwind CSS, Chart.js
* **Backend:** Java (Spring Boot), PostgreSQL
* **AI/ML Service:** Python (FastAPI/Flask), Pandas, Scikit-learn
* **DevOps:** Docker, GitHub Actions

---

## 🏗️ Project Structure
* `/ethixnode-backend`: Spring Boot application handling business logic and DB.
* `/ethixnode-ai`: Python service for currency forecasting.
* `/ethixnode-frontend`: React dashboard for user interaction.
